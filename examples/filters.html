SVG Filters

Albert can setup SVG filters. Unfortunately, there's no constraint support for filters at the moment.

The following filter nodes are supported:

  • Gaussian blur (feGaussianBlur)
  • offset (feOffset)
  • flood (feFlood)
  • composite nodes (feComposite)
  • color matrix nodes (feColorMatrix)
  • component transfer nodes (feComponentTransfer, feFuncR, feFuncG, feFuncB, feFuncA)
  • merge nodes (feMergeNode and feMerge)

See also Advanced SVG filters.
